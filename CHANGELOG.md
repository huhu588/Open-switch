# Changelog

## [0.7.0] - 2026-01-18

### 新增
- 重构 UI 组件和样式以提升设计一致性和功能性
- 更新 Tailwind 配置以支持基于 CSS 变量的主题设置
- 增强导航栏和详情面板
- 实现新的动画效果
- 优化滚动条和文本选中样式以改善用户体验

### 变更
- 初始化项目结构和配置文件
- 移除废弃的 detector 和 manager 模块

## [0.6.0] - 初始版本

### 功能
- 🔄 多 Provider 配置管理
- 🤖 模型列表管理与站点检测
- 💾 配置导出功能
- 🎨 现代化 TUI 交互界面（三栏布局）
- 📁 项目配置优先（可独立管理项目级配置）
- 🔧 MCP 服务器配置管理

### 命令
- `open-switch` - 启动 TUI 界面
- `open-switch status` - 查看当前配置状态
- `open-switch export opencode` - 导出配置到当前目录
